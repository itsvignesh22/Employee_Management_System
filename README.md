# Employee Management System (Java)

## Overview
The Employee Management System is a console-based Java application designed to manage employee records efficiently.  
It allows users to perform CRUD operations (Create, Read, Update, Delete) and generate different types of reports.

This project uses:
- Core Java
- Object-Oriented Programming concepts
- File handling with serialization
- Menu-driven console interface

---

## Features
- Add new employee
- View all employees
- Search employee by ID
- Update employee details
- Delete employee
- Generate reports:
  - Department-wise summary
  - Salary statistics
  - Employee count by position
- Save employee data to file
- Load employee data from file

## How to Run
1. Make sure Java is installed on your system.
2. Place all .java files in the same folder.
3. Compile the program:

javac *.java

4. Run the program:
java EmployeeManagementSystem

---

## Sample Output
The program displays a menu-driven interface where the user selects options by entering numbers.  
Employee data and reports are shown in a formatted tabular form.
